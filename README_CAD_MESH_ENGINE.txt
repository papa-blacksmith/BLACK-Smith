BLACK Smith Ver0.8 CAD Mesh Engine

変更:
- Ribbon Sync方式を完全廃止
- 閉じた2D輪郭ポリゴンを使用
- Three.js ShapeUtils（Earcut系）で凹ポリゴンを三角形分割
- 表面、裏面、外周側面を一体メッシュとして生成
- ギザ刃、魔剣、深い凹凸に対応
- 輪郭自己交差を2-opt方式で自動修復
- 重複頂点と不要な直線頂点を整理
- 刃先方向へ厚みを自動テーパー
- 外周へ近い部分を薄くして刃断面を表現
- 生成失敗時は前回の正常メッシュを維持
- 選択中パーツのみ3D化する安定方針を維持
- 探索、鉱物、鍛造、設計図、保管庫を維持

今回の段階:
CADポリゴン押し出し基盤です。
穴あき形状、複数輪郭、鍔・柄・ポンメル同時合成は次段階です。
