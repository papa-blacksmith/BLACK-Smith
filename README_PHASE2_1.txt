BLACK Smith Phase 2-1 Point Editing

追加:
- 空いている場所をダブルクリックして制御点追加
- Delete / Backspaceで選択点削除
- 制御点は最低3点を保護
- 点追加時に滑らかなハンドルを自動生成
- Ctrl+Z / Ctrl+Yで追加・削除・ドラッグをUndo/Redo
- 右クリックメニュー
  - 点を追加
  - 点を削除
  - スムーズ
  - シャープ
  - 左右ミラー
- 鍛造武器に自由造形データと形状IDを保存
- 保管庫で自由造形のミニプレビューを表示

注意:
既に保存済みでshapeデータがない旧武器は従来の絵文字表示になります。
